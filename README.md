FitSocial is a mobile fitness + social app built with **React Native (Expo)** and **Supabase**.

## Features

- Live jogging tracker and weekly stats
- Workout plans by muscle group
- Meals with AI calorie estimation
- Community Hub (feed, comments, challenges, allies)
- Profile with biometrics, goals and timeline

## Tech Stack

- React Native (Expo + Expo Router)
- TypeScript
- Supabase (Auth, Database, Storage, Edge Functions)
- OpenAI (meal estimator)

## Getting Started

```bash
git clone https://github.com/Bear-RSA/fitsocial.git
cd fitsocial
npm install
npm run start
